# dirty-form

> A lightweight plugin to prevent losing data when editing forms. No dependencies.

## Install

You can get it via `npm`:

```
npm install dirty-form --save
```

Or `yarn`:

```
yarn add dirty-form
```

## Setup

```javascript
import DirtyForm from 'dirty-form'

let form = document.querySelector('#form')
new DirtyForm(form)
```
